## [0.1.0] - 2024-12-04

### Added
- Initial plugin implementation
- Thompson Sampling for multi-armed bandits
- Bayesian posterior updates with Beta distributions
- Statistical significance testing with Monte Carlo simulation
- Genetic algorithm with tournament selection
- LinUCB contextual bandits
- RLHF-style preference learning
- SQLite database for experiment tracking
- Continuous learning loop with auto-pruning
- Five OpenCode plugin tools:
  - `thompson_sample` - Optimal variant selection
  - `record_feedback` - Performance tracking
  - `evaluate_ab_test` - Statistical evaluation
  - `evolve_prompts` - Genetic algorithm evolution
  - `contextual_bandit_select` - Context-aware selection
- Comprehensive README with examples
- MIT License
- TypeScript configuration
- CI/CD with GitHub Actions
- Basic test structure
